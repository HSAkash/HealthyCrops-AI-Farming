# HealthyCrops-AI-Farming-Final
This is web based machine learning project. It provide web / app service which can detect crop diseases by leaves scaning. Now it can detect 21 diseases.

## Backend:
* Django
* RestAPI
## ML:
* Tensorflow
* Tensorflow lite
* Numpy
* Pillow
## Database
* sqllite3
* PostgreSQL
## Forntend
* Html
* js
## APK
* Flutter
